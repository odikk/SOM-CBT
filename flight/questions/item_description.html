Explanation of Field Items

 

Item Label Definition
Flag Another course administrator has flagged this question for further evaluation, or to denote a recent change that warrants review.
Ques# Integer value denoting the placement of that question in the current test.
DB Rec# The integer value identifying the question as it exists in the main questions table of the database. This is THE unique identifier of the question and is independent of any tests that may contain the question.
Author The last name of the individual who can be identified as either the originator or steward of the question. For questions that were uploaded as a batch list from WebCT, this will have a default value of the director of the SOM testing labs.
Last Edited By The last name of the administrator, director, or faculty member who most recently updated or edited the question.
Allow Edit An integer value denoting whether the course administrator has granted edit privileges for supporting faculty if he/she is the author of the question. 0 = No Access 1 = Access Granted
Rating An integer value denoting the difficulty of the question as specified by the course director and/or supporting faculty. The enumerated domain ranges from 1 (easiest) to 5 (hardest).
Type An integer value denoting the question format (type). 1 = Multiple Choice 2 = Long Answer 3 = True/False
Last Edited On The date (day month year hour:minute) when the question was most recently imported or edited.
